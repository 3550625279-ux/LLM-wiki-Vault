# CAGC 思维模型

## 🧠 这是什么类型的问题？

### 问题分类

**任务类型**：多模态意图识别 (Multimodal Intent Recognition, MIR)

**问题本质**：如何从语言、视觉、听觉三种模态中准确理解人类意图？

**核心挑战**：
1. **意图歧义**：同一句话在不同场景下可能表达不同意图
2. **模态不一致**：三种模态可能给出矛盾的信号
3. **上下文依赖**：意图理解需要考虑说话者所处的场景

**问题定位**：
- 属于**多模态学习**领域
- 与**多模态情感分析**密切相关（MISA、ConFEDE 等方法）
- 是**意图识别**从单模态到多模态的自然扩展

## 🧩 如何思维化理解 CAGC？

### 类比 1：人类理解意图的过程

**场景**：你在派对上听到有人说 "I'm dying right now"

**你的理解过程**：
1. **听觉**：语气是兴奋的还是痛苦的？
2. **视觉**：表情是大笑还是皱眉？
3. **文本**：字面意思是"我要死了"
4. **上下文**：周围人是什么反应？派对氛围如何？
5. **参考经验**：在类似场景下，这句话通常是什么意思？

**CAGC 的对应**：
- 步骤 1-3 → 多模态编码器
- 步骤 4 → Cross-video Bank + CAT
- 步骤 5 → 全局对比学习

### 类比 2：图书馆查阅资料

**传统方法**：只看当前这本书
**CAGC 方法**：
1. 找到主题相似的书（场景相似度）
2. 筛选出观点一致的书（意图一致性）
3. 逐步阅读参考书（渐进式注意力）
4. 综合多本书的信息做出判断（多模态融合）

### 类比 3：法官判案

**传统法官**：只看当前案件的证据
**CAGC 法官**：
1. 查找类似案件的判例（Cross-video Bank）
2. 筛选出法律观点一致的判例（去噪）
3. 逐步参考判例（渐进式注意力）
4. 综合所有证据做出判决（多模态融合 + 对比学习）

## 📊 关键概念速查表

| 概念 | 符号 | 含义 |
|------|------|------|
| 多模态输入 | m = {L, V, A} | 语言、视觉、听觉 |
| 编码后特征 | h_t, h_a, h_v | 统一维度的模态特征 |
| 视频级特征 | v̄_i | 帧均值池化 |
| 跨视频集合 | Ω(v_i) | 场景相似的 Top-k 视频 |
| 去噪后集合 | Ω*(v_i) | 意图一致的跨视频 |
| 上下文特征 | h_m, h_cv | 跨视频上下文增强的特征 |
| 局部融合 | h_l | 文本 + 原始视觉/听觉 |
| 全局融合 | h_g | 文本 + 上下文增强的视觉/听觉 |
| 局部对比 | L_ta, L_tv | mini-batch 内的对比损失 |
| 全局对比 | L_g | 跨数据集的对比损失 |
