# CAGC 论文摘要

## 🎬 背景

**多模态意图识别 (Multimodal Intent Recognition, MIR)** 是一个新兴的研究方向，旨在通过整合语言、视觉和听觉三种模态来理解人类意图。与传统的单模态意图识别不同，MIR 面临着模态间表达不一致、意图歧义等独特挑战。

### 现有方法的局限

1. **独立建模**：现有方法（如 MAG-BERT、MulT、MISA）倾向于独立处理每个视频，忽略了视频间的全局上下文信息
2. **感知偏差**：这种独立学习方式不可避免地引入感知偏差，尤其在意图模糊的场景中
3. **模态不一致**：多模态表示的不一致性进一步放大了意图的不确定性
4. **局部对比限制**：现有对比学习方法局限于 mini-batch 内的样本级对比，无法捕获整个数据集的全局对比

---

## 🔍 问题陈述

**核心问题**：如何有效利用跨视频的全局上下文信息来缓解意图理解中的感知偏差？

**两个关键挑战**：
1. 如何捕获精细的跨视频上下文依赖关系？
2. 如何利用视频内和跨视频的上下文全局地减少模态差异？

---

## 💡 主要贡献

### 贡献 1：CAGC 方法框架
提出 Context-augmented Global Contrast (CAGC) 学习方法，从视频内和跨视频两个维度挖掘丰富的全局上下文线索。

### 贡献 2：Context-Augmented Transformer (CAT)
- 设计上下文增强 Transformer 模块，学习跨视频的上下文依赖特征
- 引入 **Cross-video Bank** 机制，通过意图倾向和视频相似度双重筛选确保跨视频源的质量

### 贡献 3：Global Context-guided Contrastive Learning (GCCL)
- 设计全局上下文引导的对比学习方案
- 将全局上下文信息作为监督信号，增强跨模态对齐
- 同时包含局部对比（mini-batch 内）和全局对比（跨整个数据集）

---

## 🏗️ 方法架构

### 整体流程

```
输入: m = {L, V, A} (语言、视觉、听觉)
        ↓
┌─────────────────────────────────────┐
│  1. 多模态编码器 (Multimodality Encoder)  │
│     - BERT (文本)                        │
│     - ResNet (视觉)                      │
│     - wav2vec (听觉)                     │
│     → CTC 对齐到统一维度                  │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│  2. 跨视频上下文学习                      │
│     - Cross-video Bank (场景相似+意图筛选) │
│     - CAT (上下文增强注意力)              │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│  3. 多模态意图解码器                      │
│     - 局部融合 (Local MF)                │
│     - 全局融合 (Global MF)               │
│     → TRM Decoder                       │
└─────────────────────────────────────┘
        ↓
┌─────────────────────────────────────┐
│  4. 全局上下文引导对比学习 (GCCL)          │
│     - 局部对比损失 L_l                   │
│     - 全局对比损失 L_g                   │
└─────────────────────────────────────┘
        ↓
输出: 意图类别
```

### Cross-video Bank 构建（两阶段）

**Stage 1 - 初始构建**：
- 对每个视频，计算视频级特征（帧均值池化）
- 用 L2 距离计算场景相似度，取 Top-k 最相似的跨视频

**Stage 2 - 去噪**：
- 通过投票原则筛选意图倾向一致的视频
- 最终得到 Ω*(v_i)，可能包含 1 到 k 个有效跨视频

### 训练目标

```
L_total = L_task + α(L_ta + L_tv) + βL_g
```

- `L_task`: 意图分类损失（交叉熵）
- `L_ta`, `L_tv`: 局部对比损失（文本-听觉、文本-视觉）
- `L_g`: 全局对比损失
- α = β = 0.02, 温度 τ = 0.7

## 📈 实验结果

### MIntRec 数据集（意图识别）

| 方法 | 20类 ACC | 20类 F1 | 二分类 ACC | 二分类 F1 |
|------|----------|---------|-----------|-----------|
| MAG-BERT | 72.65 | 68.64 | 89.24 | 89.10 |
| MulT | 72.52 | 69.25 | 89.19 | 89.07 |
| MISA | 72.29 | 69.32 | 89.21 | 89.06 |
| **CAGC** | **73.39** | **70.09** | **90.11** | **90.01** |

### CMU-MOSI 数据集（情感分析）

| 方法 | MAE↓ | Corr↑ | ACC-2↑ | F1↑ | ACC-7↑ |
|------|------|-------|--------|-----|--------|
| MISA | 0.817 | 0.748 | 82.10 | 82.0 | 41.4 |
| Self-MM | 0.708 | 0.796 | 85.46 | 85.43 | 46.67 |
| ConFEDE | 0.742 | 0.784 | 85.52 | 85.52 | 42.27 |
| **CAGC** | 0.775 | 0.774 | **85.70** | **85.60** | 44.80 |

### 消融实验（MIntRec 20类）

| 变体 | ACC | F1 |
|------|-----|-----|
| 完整 CAGC | **73.39** | **70.09** |
| 去掉 CAT | 71.31 | 67.08 |
| 去掉 L_g | 72.22 | 68.07 |
| 去掉 L_l | 72.31 | 68.33 |
| 去掉 L_g 和 L_l | 71.66 | 67.12 |

## 🔧 关键实现细节

| 参数 | 值 |
|------|-----|
| 学习率 | 2e-5 |
| Dropout | 0.5 |
| α (局部对比权重) | 0.02 |
| β (全局对比权重) | 0.02 |
| 温度 τ | 0.7 |
| Top-k* 跨视频数 | 7 |
| Mini-batch 大小 | 16 |
| 文本最大长度 | 30 tokens |
| 视频帧数 | 230 |
| 音频最大长度 | 180 |
| 隐藏维度 | 768 |
| 注意力头数 | 8 |
| 优化器 | Adam |
