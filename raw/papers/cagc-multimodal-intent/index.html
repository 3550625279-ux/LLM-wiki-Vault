Core Problem

Existing MIR methods model each video independently, ignoring global contextual information across videos. This introduces perception biases and amplifies intent uncertainty.

Example: "I am dying right now" could be Joke (positive) in an excited context or Complain (negative) in an angry context. Without cross-video context, the model cannot distinguish.

Key Innovation

CAGC captures rich global context features by mining both intra- and cross-video context interactions through:

1. CAT - Context-Augmented Transformer
2. Cross-video Bank - Quality video source retrieval
3. GCCL - Global Context-guided Contrastive Learning

How Progressive Attention Works

In each step j, the model computes attention over j cross-videos (not all at once).

Key insight: most similar videos get attended first, creating a curriculum-like learning effect.

Drag the slider to see how attention distribution changes as more cross-videos are added.

Local vs Global Contrastive Learning

Local Contrast (within mini-batch)

Limited to samples within current batch. Contrast signal is weaker due to small sample pool.

Global Contrast (entire dataset)

Global features from cross-video bank provide richer supervision. Each h_g contains cross-video context info.

Why Text as Anchor?

Research shows text modality carries the most reliable intent signal. Visual and acoustic modalities are treated as augmented versions of text. The global contrast uses text to align with cross-video global fusion features.
