论文核心思想

LLM 的 token 嵌入天然具有双曲（树状）层次结构。高频词（如 "the"、"is"）聚集在原点附近，低频词（如 "apple"、"purple"）向外扩散，形成类似"语言树"的结构。

HypLoRA 利用这一发现，直接在双曲流形上执行低秩适配，绕过传统双曲神经网络的 exp-log 抵消问题，持续优于标准 LoRA。

为什么需要双曲空间？

语言的层次结构（如"动物 → 猫 → 暹罗猫"）需要一种能自然表示"分支"的空间。欧几里得空间的体积增长是多项式的，而树状结构的节点数增长是指数的。

🔵 欧几里得空间

面积 A(r) = πr² ∝ r²

体积多项式增长 → 树的叶子节点"挤在一起"，无法充分分离

🔴 双曲空间

面积 A(r) = 2π(cosh(r)-1) ∝ eʳ

体积指数增长 → 树的叶子节点有充足空间分离，同时保持层次距离

Lorentz 双曲面模型

LKn = { x ∈ ℝn+1 : ⟨x,x⟩L = -K, x₀ > 0 }

其中 Lorentz 内积: ⟨x,y⟩L = -x₀y₀ + Σᵢ xᵢyᵢ

直观理解：这是一个嵌入在 (n+1) 维欧氏空间中的 n 维"碗状"曲面。

  • x₀（时间维度）：始终为正，控制点到"原点"的距离
  • x₁...xₙ（空间维度）：编码 token 的语义信息
  • 曲率 K：控制弯曲程度，K 越小越弯曲

δ-双曲性度量

[a,c]w ≥ min([a,b]w, [b,c]w) - δ

Gromov 乘积: [a,b]w = ½(d(a,w) + d(b,w) - d(a,b))

δ 值越小，空间越像树。论文中 LLM 嵌入的 δ ≈ 0.06-0.12，非常接近完美树 (0.0)。

抵消效应：传统方法的困境

传统双曲神经网络使用 exp/log 映射在双曲空间和切空间之间转换：

欧氏 x → exp 到双曲 → log 回切空间 → 线性变换 → exp 回双曲 → log 回欧氏

问题: exp 和 log 互逆 → 连续应用时完全抵消 → 退化为欧氏 LoRA
💡 HypLoRA 的关键洞察：不经过切空间，直接在双曲流形上操作！

LLR：直接 Lorentz 低秩变换

LLR(BA, xH) = (√(‖BAxHs‖² + K), BAxHs)

其中 xHs 是双曲点的空间分量（去掉时间分量）

验证：新点始终在双曲面上：

⟨zH, zH⟩L = -(√(‖BAxs‖²+K))² + ‖BAxs‖² = -K ✓

HypLoRA 完整流程

1️⃣
投影到双曲面
xH = ΠK exp(xE) — 将欧氏嵌入映射到 Lorentz 双曲面
↓
2️⃣
提取空间分量
xHs = xH[1:] — 去掉时间分量 xH[0]
↓
3️⃣
低秩线性变换
transformed = B·A·xHs — 标准 LoRA 变换，作用于空间分量
↓
4️⃣
重建双曲点
zH = (√(‖transformed‖²+K), transformed) — 确保输出在流形上
↓
5️⃣
投影回欧氏空间
zE = ΠK log(zH) — 输出可直接用于 Transformer 层

命题 1：高阶范数依赖

z ≈ BAx + f(‖x‖₂) · higher_order_terms(x)

由于双曲操作的非线性，HypLoRA 引入了关于输入范数 ‖x‖₂ 的高阶项：

  • 范数小的 token（高频词如 "the"）→ 高阶项贡献小 → 变换接近标准 LoRA
  • 范数大的 token（低频词如 "apple"）→ 高阶项贡献大 → 获得更丰富的变换
💡 这种自适应处理正是语言层次结构所需要的：低频/具体词需要更精细的表示调整。
