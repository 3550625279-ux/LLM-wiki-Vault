# 详细总结：Hyperbolic Fine-Tuning for Large Language Models

## 1. 背景与动机

大语言模型（LLM）如 GPT-4、LLaMA、Gemma、Qwen 在各项任务上表现出色，但它们普遍依赖**欧几里得空间**进行 token 表示。然而，语言本身具有深刻的层次结构：

- "动物" → "猫" → "暹罗猫"
- "水果" → "苹果" → "红富士"

欧几里得空间的体积增长是**多项式**的（V(r) ∝ rⁿ），这使得表示不同抽象层级的 token 时不可避免地出现拥挤和失真。双曲空间的体积增长是**指数级**的（V(r) ∝ eʳ），天然适合表示层次树状结构。

## 2. 问题定义

**核心问题**：如何在不从头训练的前提下，将双曲几何引入已有的欧几里得 LLM？

**关键挑战**：现有的双曲神经网络方法（如 Ganea et al. 2018）使用指数映射→对数映射序列来在双曲空间和切空间之间转换：

```
欧氏嵌入 → exp 映射到双曲 → log 映射回切空间 → 线性变换 → exp 映射回双曲 → 投影回欧氏
```

问题在于 **exp 和 log 是互逆操作**，连续应用时相互抵消，最终整个序列约等于原始欧氏变换 BAx，完全丧失了双曲几何的优势。

## 3. 理论分析（论文的核心贡献之一）

### 3.1 全局分析：幂律分布

对 LLaMA3-8B 在多个算术推理数据集（GSM8K、AQuA、MAWPS、SVAMP）上的 token 嵌入分析：

- **Token 频率服从幂律分布**，指数 γ ≈ 1.9
- 高频 token（如 "the"、"that"、"is"）频率 > 4900，嵌入范数小（~0.35 for LLaMA3-8B）
- 低频 token（如 "dog"、"apple"、"purple"）频率 < 300，嵌入范数大（~0.57 for LLaMA3-8B）
- **跨模型一致性**：LLaMA-7B、LLaMA-13B、LLaMA3-8B、Gemma-7B 均呈现相同模式
- 组别排序始终为 Group1（功能词）< Group2（量词）< Group3（类名词）< Group4（具体词）

### 3.2 局部分析：δ-双曲性

使用 Gromov 四点条件计算每个 prompt 中 token 嵌入的 δ-双曲性：

| 度量空间 | δ |
|---------|-----|
| 球面空间 | 0.99±0.01 |
| 随机图 | 0.62±0.34 |
| PubMed 图 | 0.40±0.45 |
| 无标度图 | 0.00 |
| 树图 | 0.00 |

| 模型 | MAWPS | SVAMP | GSM8K | AQuA |
|------|-------|-------|-------|------|
| LLaMA-7B | 0.08±0.02 | 0.09±0.01 | 0.10±0.01 | 0.10±0.01 |
| LLaMA-13B | 0.08±0.01 | 0.09±0.01 | 0.09±0.01 | 0.10±0.01 |
| Gemma-7B | 0.11±0.01 | 0.11±0.01 | 0.11±0.01 | 0.12±0.01 |
| LLaMA3-8B | 0.06±0.01 | 0.07±0.01 | 0.07±0.01 | 0.08±0.01 |

**结论**：LLM token 嵌入的 δ 值接近 0（完美树），远低于球面空间（~1），表明嵌入空间具有显著的树状层次结构。

### 3.3 理论桥梁：幂律 ↔ 双曲几何

在 Poincaré 圆盘模型（曲率 K=-1）中：
- 圆的周长和面积呈指数增长：C(r) = 2π sinh(r) ≈ eʳ
- Token 径向分布 p(r) ∼ e^(-ζr)
- 频率函数 k(r) ∼ e^(-r)
- 推导得 P(k) ∼ k^(-(1-ζ))，即幂律分布
- 幂律指数 γ = 2/ζ + 1

## 4. HypLoRA 方法

### 4.1 直接 Lorentz 低秩变换（LLR）

**核心创新**：不通过切空间，直接在双曲流形上执行低秩线性变换。

给定输入 x_E ∈ ℝ^d：
1. 投影到双曲空间：x_H = Π_K exp(x_E)
2. 直接 Lorentz 低秩变换：

```
LLR(BA, x_H) = (√(‖BA x_H^s‖² + K),  BA x_H^s)
```

其中 x_H^s 是 x_H 的空间分量（去掉时间分量 x_H[0:1]）。

3. 投影回欧氏空间：z_E = Π_K log(x_H')

### 4.2 关键性质（命题 1）

由于双曲操作的非线性，HypLoRA 引入的等效变换包含关于输入嵌入 x 的**高阶项**，且这些项显式依赖于 x 的 L2 范数 ‖x‖₂。这意味着：
- 高频 token（范数小）和低频 token（范数大）会被**差异化处理**
- 自然对齐于嵌入空间的层次结构

### 4.3 时间复杂度

与标准 LoRA 相同：O(r·(d+k))。额外的流形映射操作可在 O(N) 内完成（N 为 token 数）。

### 5.3 曲率敏感性

在 Gemma3-4B 上，K=0.5 在算术推理上最优（M.AVG 77.8 vs K=1.0 的 75.8），常识推理上 K=1.0 在部分模型（LLaMA3-8B、Qwen2.5-7B）上表现更好。

### 5.4 效率

所有模型可在约 1 小时内完成微调。HypLoRA 的推理开销高于 LoRA，但低于 DoRA。

## 6. 作者承认的局限

1. 分析主要在算术推理和常识推理数据集上，代码/多语言/长上下文泛化性未验证
2. Token 频率与嵌入范数的关系是相关性而非因果性
3. δ-双曲性在 prompt 级别测量，语料库级别可能有不同结论
4. 额外的双曲投影带来适度推理开销
