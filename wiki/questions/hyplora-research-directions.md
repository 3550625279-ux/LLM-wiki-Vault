# HypLoRA 研究方向与开放问题

> 基于论文阅读后的深度思考，识别出 5 个有潜力的研究方向。

---

## 方向 1：动态曲率 K

**现状**：HypLoRA 的曲率 K 是固定的或全局可学习的。

**问题**：不同层、不同 token 可能需要不同的"弯曲程度"。如 Transformer 的浅层处理局部语法（可能不需要太多双曲结构），深层处理语义推理（可能需要更强的双曲性）。

**研究方向**：让 K 依赖于层深度或 token 位置，或基于 δ-双曲性的注意力值在每层学习动态曲率。

**预期**：进一步提升模型灵活性，尤其在复杂推理任务上。

---

## 方向 2：多头曲率

**现状**：每个注意力头独立工作，但共享全局 K。

**问题**：不同注意力头可能关注不同类型的关系（语法、语义、位置等），这些关系可能对应不同的层次结构深度。

**研究方向**：为每个注意力头配备独立的曲率 Kₕ，让模型自动学习每个头的最优几何特性。

**预期**：不同头的 K 分化可能揭示模型内部的"几何分工"。

---

## 方向 3：ResNet 类跳跃连接

**现状**：HypLoRA 是纯前馈的双曲变换。

**问题**：深层网络中，纯双曲变换可能丢失一些原始欧几里得空间中的信息。

**研究方向**：在 HypLoRA 中添加跳跃连接，将原始欧几里得嵌入与双曲变换后的嵌入融合。

```
z_final = α · z_euclidean + (1-α) · z_hyperbolic
```

**预期**：保留两种几何的互补优势，可能提升泛化性。

---

## 方向 4：HypLoRA 与 AFT 结合

**问题**：论文提出了 AFT（注意力头级融合），但未探索与 HypLoRA 的结合。

**研究方向**：在双曲空间中进行注意力头级融合，利用树状结构指导融合策略。

---

## 方向 5：几何自适应机制

**问题**：HypLoRA 的双曲假设是固定的——它假设所有嵌入都是双曲的。但实际数据可能混合了层次和平坦结构。

**研究方向**：让模型自动学习何时使用双曲变换、何时使用欧氏变换，基于输入数据的局部几何特性。

---

## 关键公式

### 核心数学工具

| 公式 | 含义 |
|------|------|
| ⟨x,y⟩_L = -x₀y₀ + Σᵢ xᵢyᵢ | Lorentz 内积 |
| x₀ = √(‖x‖² + K) | 欧氏到双曲面投影 |
| d(x,y) = acosh(-⟨x,y⟩_L / K) | 双曲距离 |
| δ-双曲性 ≈ 0.1 | LLM 嵌入接近树的程度 |
| γ = 2/ζ + 1 | 幂律指数与曲率关系 |
| z_H = (√(‖BAx‖²+K), BAx) | LLR 直接 Lorentz 变换 |

## 理论限制

- 嵌入空间不是"完全"双曲（δ > 0），假设过于理想化
- 基于双曲几何的神经网络训练复杂度较高
- 模型压缩时存在精度损失风险
- δ ≈ 0.1 的结论局限于算术推理数据集，泛化性存疑
