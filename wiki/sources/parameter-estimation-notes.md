# 参数估计学习笔记

## 概述

用户个人的参数估计学习笔记，涵盖从点估计到区间估计的完整知识体系。

## 核心知识点

1. **点估计** — 用一个数值估计未知参数，估计量本身是随机变量
2. **矩估计** — 用样本矩估计总体矩，本质是频率稳定于概率
3. **极大似然估计（MLE）** — 最可能产生观测数据的参数值
4. **顺序统计量** — 样本排序后的统计量，极值分布
5. **估计量评价** — 无偏性、有效性、相合性三大标准
6. **区间估计** — 枢轴量法，四种分布情形（正态/卡方/t/F）
